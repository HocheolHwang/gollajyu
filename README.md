# 골라쥬 🏦

💰 본 프로젝트는 삼성 청년 SW 아카데미 2학기 공통 프로젝트 결과물입니다.





## 개요🌱

- 현대인 한명이 하루에 접하는 정보량은 20세기 초 사람들이 평생 접한 정보량에 버금갈 정도
- 수많은 정보는 선택의 다양성을 넓혔다
- 하루에도 수 천 가지의 선택을 해야하는 우리를 위한 웹 어플리케이션
- 내 선택을 골라쥬~
